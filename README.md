# ESP32C3
notes on using ESP32C3

## C API

ensure that python is configured so that `python` runs `python3`:

```
sudo ln -s /Library/Developer/CommandLineTools/usr/bin/python3 /Library/Developer/CommandLineTools/usr/bin/python
```

Then:

```
cd ~/esp/esp-idf
# or 
# ./install.sh
source export.sh
```
